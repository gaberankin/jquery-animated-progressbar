Click!     

The above is an implementation of an animated progress bar similar to the interface widget common to most operating systems. while this doesn't really have much application in a website environment, I was bored and decided to whip it up (it took about two hours). It was either this or watch Steven Segal movies.

It was made as a jQuery plugin, so it's fairly easy for other developers to add to their own sites. the HTML code creating this particular control is:

<div id="bar"></div>
<a href="#" id="clickTrigger">Click!</a>
&nbsp;&nbsp;&nbsp;&nbsp;<span id="dots"></span>

And the javascript code is:

$(document).ready(function(){
	$('#bar').loading({
		duration: 2000,
		width: 500,
		height: 3,
		color: '#000088',
		border: '1px solid #000088',
		padding: '1px',
		displayPercent: true,
		clickTrigger: '#clickTrigger',
		onBegin: function(){ $('#clickTrigger').text('starting!'); },
		onComplete: function(){ $('#clickTrigger').text('Done!'); },
		specialPercents: [25,50,75,100],
		onSpecialPercent: function(curPercent)
						{
							var curDots = $('#dots').text();
							$('#dots').text(curDots+'.');
						}
		});
});

Some notes:

  • jQuery 1.4+ is required (1.7.2 included in repository)
  • there is a bug with the 'specialPercents' & 'onSpecialPercent' functionality. note the code above. it is meant to add a dot after the start button when it reaches 25%, 50%, 75%, and 100%, meaning that by the time the progress bar reaches the end there should be 4 more dots in the list. This is not always the case - sometimes it only adds 3. This is due to a rounding issue that I may or may not look into later depending on how bored I am
